# Проект: Место
Проект представляет из себя интерактивную страницу, куда можно добавлять фотографии, удалять их и ставить лайки.

## Технологии
* React
* JavaScript
* Адаптивная верстка

## Запуск проекта
1. Клонировать репозиторий:
`git@github.com:elmanAliev/mesto-react.git`

2. Установить зависимости:
`npm install`

3. Запуск проекта:
`npm start`
